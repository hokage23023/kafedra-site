# Проект сторінки кафедри

Проект створено за допомогою [Create React App](https://github.com/facebook/create-react-app).

## Каталоги та їх призначення

### `components`

Усі реакт-компоненти.

#### `ui`

Загальні компоненти, що відповідають за зовнішній вигляд сайту.

#### `pages`

Компоненти, які об'єднують інші у повноцінні сторінки.

#### `layout`

Компоненти, що повторються, типу хедера та футера.

---

### `assets`

Каталог, що відповідає за зовнішній вигляд.

#### `images`

Усі картинки, та інші медіа.

#### `styles`

CSS стилі застосунку.

---

### `services`

Функції отримання JSON файлів з даними для сайту.

---

### `data`

Каталог з даними у форматі JSON замість бекенду.

## Доступні скрипти

### `npm start`

Запустить локальний сервер\
За адресою [http://localhost:3000](http://localhost:3000)

### `npm run build`

Створить готовий до деплойменту проект у каталозі `build`.
